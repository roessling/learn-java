Konfiguration

Die Konfiguration des Programmes muss vorgenommen werden, wenn es erstmalig auf einem Gerät ausgeführt wird. Nach dem erstmaligen Start des Programmes wird eine leere Konfigurationsdatei erzeugt. Um diese mit den richtigen Werten zu füllen, werden Sie aufgefordert, zwei Pfade einzugeben.

JDK

Der erste Pfad führt zum JDK-Ordner. Das JDK (Java Development Kit) muss hierfür installiert sein. Sollten Sie kein JDK installiert haben oder Ihr JDK älter als Version 1.8.0_131 sein, funktioniert das Programm nicht oder nur teilweise. Herunterladen können Sie das JDK auf dieser Seite. Sollte sich der Link nicht öffnen, benutzen Sie Rechtsklick -> "Link in Zwischenablage kopieren" und öffnen Sie die Seite in Ihrem Browser, indem Sie den Link dort in die Adresszeile einfügen.

Sobald das JDK installiert ist, tragen Sie den Ordner in das entsprechende Feld ein. Dafür können Sie entweder den Installationspfad kopieren und einfügen, oder Sie verwenden den Knopf "Durchsuchen...", um einen Dateibrowser zu öffnen. Unter Windows mit der 64bit-Version des JDK 1.8.0_131 sieht der Pfad wie folgt aus.

Achten Sie darauf, dass der angegebene Ordner die Unterordner "bin" und "jre" enthält. Es ist nicht ausreichend, auf den Ordner "Java" zu verweisen. Eine entsprechende Fehlermeldung erscheint im Reiter "Compiler" des Programms, wenn Sie versuchen, ein Programm zu übersetzen, ohne dass der JDK-Pfad korrekt gesetzt ist.

JUnit

Der zweite Pfad führt zu einer .jar-Datei, die die JUnit-Bibliothek enthält. Diese Datei wird typischerweise mit dem Programm heruntergeladen und muss nur noch verlinkt werden. Angenommen, Sie haben das Programm unter Windows in den Ordner C:\Programme (x86)\JavaAufgaben\ entpackt, so würde der Pfad zu JUnit so aussehen.

JUnit wird für die Kompilierung und Ausführung der Tests benötigt. Das Programm wurde mit JUnit Version 4.12 entworfen und getestet. Abweichungen der Funktionalität bei anderen Versionen sind möglich.
